gwpopulation.conversions.alpha_beta_max_to_mu_var_max

gwpopulation.conversions.alpha_beta_max_to_mu_var_max(alpha, beta, amax)[source]

Convert between parameters for beta distribution

\[ \begin{align}\begin{aligned}\mu &= a_\max \frac{\alpha}{\alpha + \beta}\\\sigma^2 &= a_\max^2 \frac{\alpha\beta}{(\alpha + \beta)^2 + (\alpha + \beta + 1)^2}\end{aligned}\end{align} \]
Parameters:
alpha: float

The Beta alpha parameter (\(\alpha\))

beta: float

The Beta beta parameter (\(\beta\))

amax: float

The maximum value (\(a_\max\))

Returns:
mu: float

The mean (\(\mu\))

var: float

The variance (\(\sigma^2\))

amax: float

The maximum spin (\(a_\max\))
