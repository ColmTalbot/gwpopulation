gwpopulation.conversions.mu_var_max_to_alpha_beta_max¶

gwpopulation.conversions.mu_var_max_to_alpha_beta_max(mu, var, amax)[source]¶

Convert between parameters for beta distribution

\[ \begin{align}\begin{aligned}\alpha &= \frac{\mu^2 (a_\max - \mu) - \mu \sigma^2}{a_\max\sigma^2}\\\beta &= \frac{\mu (a_\max - \mu)^2 - (a_\max - \mu) \sigma^2}{a_\max\sigma^2}\end{aligned}\end{align} \]
Parameters:
mu: float

The mean (\(\mu\))

var: float

The variance (\(\sigma^2\))

amax: float

The maximum value (\(a_\max\))

Returns:
alpha: float

The Beta alpha parameter (\(\alpha\))

beta: float

The Beta beta parameter (\(\beta\))

amax: float

The maximum spin (\(a_\max\))
