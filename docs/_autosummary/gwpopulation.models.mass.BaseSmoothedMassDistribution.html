static smoothing(masses, mmin, mmax, delta_m)[source]

Apply a one sided window between mmin and mmin + delta_m to the mass pdf.

The upper cut off is a step function, the lower cutoff is a logistic rise over delta_m solar masses.

See T&T18 Eqs 7-8 Note that there is a sign error in that paper.

S = (f(m - mmin, delta_m) + 1)^{-1} f(m’) = delta_m / m’ + delta_m / (m’ - delta_m)

See also, https://en.wikipedia.org/wiki/Window_function#Planck-taper_window
