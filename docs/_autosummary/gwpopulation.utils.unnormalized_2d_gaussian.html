gwpopulation.utils.unnormalized_2d_gaussian

gwpopulation.utils.unnormalized_2d_gaussian(xx, yy, mu_x, mu_y, sigma_x, sigma_y, covariance)[source]

Compute the probability distribution for a correlated 2-dimensional Gaussian neglecting normalization terms.

\[\begin{split}\ln p(x) = (x - \mu_{x} y - \mu_{y}) \Sigma^{-1} (x - \mu_x y - \mu_{y})^{T} \\ \Sigma = \begin{bmatrix} \sigma^{2}_{x} & \rho \sigma_{x} \sigma_{y} \\ \rho \sigma_{x} \sigma_{y} & \sigma^{2}_{y} \end{bmatrix}\end{split}\]
Parameters:
xx: array-like

Input data in first dimension (\(x\))

yy: array-like

Input data in second dimension (\(y\))

mu_x: float

Mean in the first dimension (\(\mu_{x}\))

sigma_x: float

Standard deviation in the first dimension (\(\sigma_{x}\))

mu_y: float

Mean in the second dimension (\(\mu_{y}\))

sigma_y: float

Standard deviation in the second dimension (\(\sigma_{y}\))

covariance: float

The normalized covariance (bounded in [0, 1]) (\(\rho\))

Returns:
prob: array-like

The unnormalized probability distribution (\(p(x)\))
