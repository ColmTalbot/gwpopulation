gwpopulation.utils.von_mises

gwpopulation.utils.von_mises(xx, mu, kappa)[source]

PDF of the von Mises distribution defined on the standard interval.

\[p(x) = \frac{\exp\left( \kappa \cos(x - \mu) \right)}{2 \pi \mathcal{I}_{0}(\kappa)}\]
Parameters:
xx: array-like

Input points at which to evaluate the distribution

mu: float

The mean of the distribution

kappa: float

The scale parameter

Returns:
array-like

The probability

Notes

For numerical stability, the factor of exp(kappa) from using i0e is accounted for in the numerator
